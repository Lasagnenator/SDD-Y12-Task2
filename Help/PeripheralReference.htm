Address Device Infromation
0000 Graphics

Write the X coordinate of the pixel.

The screen is 127x127 so only bits 0-6 will actually be used.

Only writable.
0001 Graphics

Write the Y coordinate of the pixel.

The screen is 127x127 so only bits 0-6 will actually be used.

Only writable.
0002 Graphics

Write the colour value of the pixel

RGB with bits 0-4 being red, 5-11 being green, 12-15 for blue.

Only writable. When colour value is written, the screen gets updated with the new x,y,colour pixel.
0010 Push Buttons

Read the state of the 8 buttons.

Bits 0-7 are the states of each of the buttons. 0 means unpressed and 1 means pressed.

Only read from. Writing does nothing.
0020 LED Output

Controls the 8 LEDs.

Bits 0-7 are the states of each of the LEDs. 0 means off and 1 means on.

Only writable. Reading returns zero.
0030 Timer

A timer that increments its read value by one every 50 milliseconds and wraps around to zero when it goes above 65535 ticks.

Only readable. Writing does nothing.

 

It should be noted that a peripheral is not limited to a single peripheral address and can take multiple as seen with the graphics peripheral. It is up to the designer of the peripheral as to how it should work across multiple addresses and it is up to the programmer to make sure that they use the peripheral correctly.
