Building

Assembler

Requirements:

  • Python 3.7 or later
  • Pyinstaller 3.4 or later
  • WxPython 4.0.3 or later

Use the included build-pyinstaller.bat. After running and if there are no errors, the executable folder will be created in the dist folder.

Simulator

Requirements:

  • Visual Studio

Open Simulator.sln in Visual Studio.

Change the mode to Release. (Debug if you need to step through the code. The code has been optimised for Release mode so you should use that more).

Build the solution.
