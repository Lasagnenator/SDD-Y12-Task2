Simulator Usage

  1. Open Simulator.exe (Either run the incuded executable or build it yourself)
  2. Open a .r16.asm binary file. This can be done by using the File menu and selecting Open or using the keyboard shortcut Ctrl+O.
    If the binary file is malformed, the simulator will tell you that it is malformed.
  3. Press the Reset button. This will load the program into memory and reset all the peripherals to their original state.
  4. To step through the program, use the Step button. This will automatically highlight the current program counter in memory.
  5. To run the program at full speed, use the Power On button.
  6. The Power Off and Pause buttons allow you to stop execution. Pausing allows you to resume execution while Powering Off means that you need to Start from the beginning again.
  7. When Paused, the machine becomes available for stepping through instructions.
